About Groupr


After signing up, you can create a basic profile and create an Event. An "Event" is any large group of people that you would like to form subgroups of. For example, a classroom or a Hackathon Event. The owner of the Event will be given an invite code. Once users have joined the event, they will be able to see other users, and start choosing if they want to work with them. Groups will be formed and can be discarded if the group decides to do so.



For questions or inquiries, please email usegroupr@gmail.com
